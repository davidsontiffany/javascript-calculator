# JavaScript Calculator

A simple calculator built using HTML, CSS, and JavaScript that performs basic arithmetic operations with a clean and responsive user interface.

## Features
- Addition, subtraction, multiplication, and division
- Decimal input support
- Clear and delete functionality
- Error handling for division by zero

## Technologies Used
- HTML
- CSS
- JavaScript

## What I Learned
This project helped me understand DOM manipulation, event handling, and managing application state in JavaScript. I also practiced debugging and handling edge cases like multiple decimals and invalid operations.

## Future Improvements
- Add keyboard support
- Improve UI styling for different button types
